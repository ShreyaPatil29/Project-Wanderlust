# Project Wanderlust 🌍

A full-stack travel stay booking web application inspired by Airbnb.

## Features 🚀

- User Signup / Login
- Add Listings
- Search Destinations
- Booking System
- Check Availability
- Payment Method
- Reviews & Ratings

## Tech Stack 💻

- Node.js
- Express.js
- MongoDB
- EJS
- Bootstrap
- CSS
- JavaScript

## How to Run ⚙️

1. Clone repository
2. npm install
3. Add .env file
4. node app.js
